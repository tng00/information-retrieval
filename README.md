# Информационный поиск: Реализация поисковой системы

Репозиторий содержит лабораторные работы по курсу "Информационный поиск". Целью проекта является последовательная разработка всех компонентов поисковой системы с нуля, от сбора корпуса документов до реализации веб-интерфейса для булева поиска.

Основное ядро системы (индексатор, поисковик) написано на C++. Вспомогательные скрипты для сбора данных, анализа и веб-интерфейса реализованы на Python.

## Структура проекта

*   **[Лабораторная работа №1: Добыча корпуса документов](https://github.com/tng00/information-retrieval/tree/main/lab1)**
    *   Анализ источников данных.
    *   Сбор и изучение характеристик корпуса научных статей с портала CyberLeninka.

*   **[Лабораторная работа №2: Поисковый робот](https://github.com/tng00/information-retrieval/tree/main/lab2)**
    *   Реализация краулера на Python.
    *   Использование MongoDB для хранения очереди URL и скачанных документов.

*   **[Лабораторная работа №3: Токенизация и стемминг](https://github.com/tng00/information-retrieval/tree/main/lab3)**
    *   Разработка токенизатора на C++ для разбиения текста на лексемы.
    *   Реализация стеммера Портера для русского языка.
    *   Анализ частотного распределения по закону Ципфа.

*   **[Лабораторная работа №4: Построение булева индекса](https://github.com/tng00/information-retrieval/tree/main/lab4)**
    *   Реализация индексатора на C++, строящего прямой и обратный индексы.
    *   Разработка собственного бинарного формата для хранения словаря, списков вхождений (с d-gaps) и данных о документах.

*   **[Лабораторная работа №5: Булев поиск](https://github.com/tng00/information-retrieval/tree/main/lab5)**
    *   Реализация поискового движка на C++, поддерживающего операции `AND`, `OR`, `NOT` и скобки.
    *   Создание веб-интерфейса на Flask для взаимодействия с поисковой системой.
